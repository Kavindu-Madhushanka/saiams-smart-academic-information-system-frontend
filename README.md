# SAIAMS Frontend

## Overview

This repository contains the **React.js frontend application** for the
**Smart Academic Information and Attendance Monitoring System (SAIAMS)**.

The frontend delivers a fast, responsive, and user-friendly dashboard
for real-time academic data visualization.

---

## Technology Stack

- React.js
- Material-UI (MUI)
- REST API integration
- Responsive Single Page Application (SPA)

---

## Features

- Student dashboard (attendance, ICA marks, GPA)
- Lecturer dashboard (manage records, respond to issues)
- Admin dashboard (user management)
- Notifications & alerts
- Secure login & session handling
- Mobile-friendly UI
